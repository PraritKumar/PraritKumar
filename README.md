## 💼 What I've Built

- 🏛️ **National Tax Ingestion System (Infosys)** — Migrated India's legacy file-based tax system to a low-latency service-oriented architecture using Java 21 + Spring Boot. Processes **511K+ daily transactions** at **52 TPS** within a **3-second SLA**. Cut bulk processing time by **60–70%** via JVM-level multithreading and tuned thread-pool executors. Secured with OAuth 2.0 + RBAC for government-grade compliance.

Integrated Redis caching, improving API response times by **25–40%**. Maintained **80%+ code coverage** via JaCoCo with SonarQube quality gates.
